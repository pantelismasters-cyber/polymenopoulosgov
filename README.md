# polymenopoulos.gov — static demo site

Αυτό το repository περιέχει ένα απλό, στατικό site που μιμείται το layout του αρχικού demo. Για να το προσαρμόσετε πλήρως με τις φωτογραφίες που στείλατε, αντιγράψτε τα αρχεία εικόνων σας μέσα στο φάκελο `assets/images` με τα ονόματα `photo1.jpg` και `photo2.jpg` (ή `photo1.png` / `photo2.png`) και αντικαταστήστε τα placeholder SVG αρχεία.

Βήματα ανάπτυξης (GitHub Pages):

1. Τοπικά — προεπισκόπηση ανοίγοντας `index.html` σε browser.
2. Commit & push στο GitHub:

```bash
git add .
git commit -m "Initial polymenopoulos.gov site"
git push origin main
```

3. Ενεργοποιήστε GitHub Pages στο repository settings και επιλέξτε branch `main` (root). Το αρχείο `CNAME` ορίζει το όνομα `polymenopoulos.gov`.

Σημειώσεις:
- Αν θέλετε, μπορώ να τοποθετήσω εγώ τις εικόνες που στείλατε αν μου επιβεβαιώσετε ότι επιτρέπετε να προσθέσω τα αρχεία απευθείας στο repo.
